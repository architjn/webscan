# webscan

Steps To Run the project -
1. `npm install`
2. `node index.js`

Project will start on hardcoded port `3000`.

You can visit API - http://localhost:3000/api/url?page=https://npmjs.com/

Where `page` is query parameter, that can be any URL to process.
